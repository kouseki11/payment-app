
# Payment app

A brief description of what this project does and who it's for


## Features

- Deposit
- Withdrawal
- Payment
- Transaction History


## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

### Third Party Folder
`PORT`
`DB_HOST`
`DB_USER`
`DB_PASSWORD`
`DB_NAME`
`DB_DIALECT`

### Backend Folder
`PORT`
`FULL_NAME`
## Installation

Install payment-app with npm open 3 terminal
- First set-up your env

```bash
  cd third_party
  npm install
  npm run migrate
  npm run seed
  npm start
```
```bash
  cd backend
  npm install
  npm run swagger
  npm start
```
```bash
  cd frontend
  npm install
  npm run dev
```
